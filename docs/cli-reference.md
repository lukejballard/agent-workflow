# CLI Reference

## Current State

This repository snapshot does not include a validated public command-line interface for the
baseball-biomechanics-ai product.

## What Exists Today

The workspace does include helper and maintenance scripts under `scripts/`, but those are
repo-support utilities rather than a documented end-user CLI surface.

Examples include:
- agent-platform maintenance helpers
- local repository utilities
- legacy migration or demo placeholders inherited from earlier project history

## Guidance

Do not treat any inherited command examples from older drafts as supported product
commands unless the runtime implementation is restored and the commands are revalidated.

A future CLI reference should only document commands that are:
1. backed by checked-in runtime code
2. covered by tests
3. aligned with the biomechanics product model
