# Assessment Workspace Guide

This file replaces an inherited "Pipeline Studio" guide that no longer matches the
current product direction.

## Current Status

The current workspace snapshot does not contain a verified authoring or orchestration
workspace for baseball-biomechanics-ai. There is no validated backend or frontend
surface in this repo snapshot that supports creating, running, or reviewing
assessment workflows end to end.

## What This Page Represents Now

Use this page as a placeholder for the future practitioner-facing workspace that may
support tasks such as:
- reviewing athlete sessions
- comparing assessments over time
- capturing practitioner notes
- drafting movement prescriptions
- exporting shareable reports

## What Is Intentionally Not Documented

This page intentionally does not describe:
- API endpoints
- orchestration flows
- DAG or node editors
- runtime execution targets
- authentication contracts

Those details belong in a future product spec only after the real runtime surfaces are
restored in the repository.
