# Pipeline Graph Screenshot

This is a placeholder for the **Pipeline Graph (DAG Viewer)** screenshot.

The pipeline graph shows:
- Each pipeline step as a node
- Step dependencies as directed edges
- Step status colour-coding (success / failed / running)
- Click-through to step-level event details

## To replace this placeholder

1. Start the full stack: `docker compose up -d`
2. Run the demo pipeline: `python scripts/run_demo_pipeline.py`
3. Open `http://localhost:3000` and navigate to the **DAG Viewer** tab
4. Take a screenshot and save it as `docs/images/pipeline_graph.png`
5. Remove this markdown file
